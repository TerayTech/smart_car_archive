# SmartCar_Motherboard_MK66_4layers
使用MK66主控的智能车磁车主板  
本项目为我组准备参加15届智能车比赛所使用的主板，但是因为一些原因现在并不打算继续进行下去了。现在上传开源供大家学习参考使用。  
总的来说设计没有问题，运行稳定。无线串口的接口是7pin的。如果有啥问题，可以B站私信。 ID:TerayTech
